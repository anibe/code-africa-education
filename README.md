# Simple SMS Eduction

## Services used

* [Twilio](http://twilio.com) for SMS messaging and WebHook SMS receipt notification
* [PHPFog](http://phpfog.com) for hosting
* [Pusher](http://pusher.com) for realtime push notifications to the dashboard
